================================================================================
                    TROUBLESHOOTING
================================================================================

PROBLEM: "Streamlit not found"
SOLUTION: pip install streamlit

PROBLEM: Models not loading
SOLUTION: Run: python fraud_detection_model.py

PROBLEM: Port 8501 already in use
SOLUTION: streamlit run streamlit_app.py --server.port 8502

PROBLEM: Batch file won't run
SOLUTION: Try: cmd /k "streamlit run streamlit_app.py"

PROBLEM: Very slow performance
SOLUTION: Close other apps, ensure 4GB+ RAM available

PROBLEM: CSS/UI looks broken
SOLUTION: Clear browser cache (Ctrl+Shift+Delete) and refresh
